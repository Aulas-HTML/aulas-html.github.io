Sobre o Curso

A linguagem de marcação de hipertexto (Hypertext Markup Language - HTML) é a base de qualquer website e de muitos aplcativos moveis. Sendo o HTML a linguagem mais utilisada na internet, se tornar fluente lhe fornece a capacidade de construir diversos tipos de contúdos para diversos tipos de dispositivos conectados a web.

O objetivo deste curso é fornecer base para começar a sua jornada na fluência do HTML, abordaremos os conceitos básicos da linguagem, aqueles que muitas vezes são ignorados no processo de aprendisagem por acreditar que nada deve ser deixado de lado. No decorrer do curso iremos sempre fazer links a diversas outrasa tecnologias que estão trabalhando ao lado do HTML, bem como o protocolo de transferência de hipertexto (HyperText Transfe Protocol - HTTP) a qual esta alicerçada toda a Web, bem como algumas outras linguagens como CSS (Cascading Style Sheets) que permite estilizar suas pâginas ou seja, deixalas bonitas e a Linguagem JavaScript a qual poderá ser usada para determinar certos comportamentos a sua página de internet ou melhor, seu WebSite.

Um outro fator importante é que este curso tem foco na versão 5 da linguage, o HTML5.

O que este curso não fornece

  • Este curso não vai tratar sobre construir Layouts

    • Embora vocês será o tempo todo estimulado a usar sua criatividade para construir seus layouts e também verá alguns layouts que podem ser classificados como bons modelos.

  • Não é um curso de CSS

    • Mesmo que possa parecer estarmos vendo bastante sobre CSS (Não estamos!), CSS não é o objeto deste curso.

  • Não é um curso de JavaScript

    • Por mais que tenhamos que escrever código JavaScript (JS) em alguns momentos, utilizamos mesma ideia aplicada ao CSS, JS não é o objetivo deste curso.

Como veremos o conteúdo agregado

Como vá mencionado, diversos outros conteúdos serão inseridos dentro do curso de HTML, embora eles sejam bem peculiares, iremos abordar de forma bem simples, sempre implementando apenas o necessário para que a complexidade do curso fique apenas no HTML.

Ementa do Curso

A ementa programada para este curso é totalmente mutacional, de acordo com o retorna dado pelas pessoas que estão fazendo o curso, as aulas podem receber upgrades ou inserir novas partes. As aulas são preparadas pensando sempre em um formato que alicerce a base do conhecimento.
